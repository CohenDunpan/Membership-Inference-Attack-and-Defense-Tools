# Membership-Inference-Attack-and-Defense-Tools
This project is an open-source research toolkit for studying privacy inference attacks and defense mechanisms in machine learning models. Motivated by the severe risks of membership privacy leakage in ML models, the toolkit integrates existing attack and defense methods to support systematic evaluation and benchmarking of privacy risks.

本分叉将在少量MIA攻防算法实现的基础上，开发面向长期演化的整体架构，并为后续更多MIA算法注册接入以及Agent接入维护打下基础。
## 项目简介
本项目是一个开源的研究工具包，旨在研究机器学习模型中的成员推断攻击和防御机制。
项目旨在实现现有的各类MIA攻防算法，提供统一接口，支持对隐私风险的系统评估和基准测试。
## 分叉目的
本分叉针对项目初期架构尚不明晰，缺乏长期维护和扩展能力的问题，进行架构重构和功能完善。
目标是设计一个模块化、可扩展的整体架构，便于后续更多MIA算法和Agent的注册接入与维护。
## 主要改动
1. 设计并实现模块化架构，定义清晰的接口规范。
2. 实现基础的MIA攻击和防御算法，作为后续扩展的基础。
3. 提供注册机制，方便后续添加更多MIA算法和Agent。
4. 编写详细的文档，指导用户如何使用和扩展工具包。
## 未来计划
1. 持续添加更多MIA攻击和防御算法。
2. 优化现有算法的性能和效果。
3. 开发用户友好的界面，提升使用体验。
4. 建立社区，促进研究人员之间的交流与合作。
